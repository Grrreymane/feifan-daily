### 1. Simon Willison: HTML 的不合理有效性 — Using Claude Code

**链接**：[simonwillison.net](https://simonwillison.net/2026/May/8/unreasonable-effectiveness-of-html/) | [HN 345票](https://news.ycombinator.com/item?id=48071940)

Simon Willison 引用了 Anthropic Claude Code 团队 Thariq Shihipar 的文章，提出一个颠覆性观点：**用 HTML 替代 Markdown 作为 Claude 的输出格式**。文章包含大量实例和 prompt 建议，论证了 HTML 在 AI 辅助编程中的独特优势——Claude 生成的 HTML 可以直接在浏览器中渲染为交互式原型、可视化报告和丰富的文档格式，省去了 Markdown → HTML 的转换步骤。Simon 还分享了自己在实践中的具体工作流改变。

**为什么值得读**：如果你日常使用 Claude Code 或其他 AI 编码工具，这篇文章可能直接改变你的工作流。作为 AI 工程领域最有影响力的独立博主之一，Simon 的实践验证极具参考价值。
