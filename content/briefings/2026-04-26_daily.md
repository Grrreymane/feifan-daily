# 🔥 今日深度推荐 Deep Reads

## 1. DeepSeek V4 Preview — 开源 AI 的新里程碑

**来源**：[DeepSeek 官方](https://api-docs.deepseek.com/news/news260424) | [HN ⬆️2009](https://news.ycombinator.com/item?id=47884971)

DeepSeek 本周正式上线 V4 预览版并同步开源，这是开源大模型领域的重大突破。

**核心亮点**：
- 双模型架构：V4-Pro（1.6T 总参/49B 激活）+ V4-Flash（284B 总参/13B 激活）
- **1M 上下文成为默认标准**，384K 最大输出长度
- V4-Pro 的 FLOPs 仅为 V3.2 的 27%，KV cache 降至 10%
- Flash 定价 $0.14/1M 输入、$0.28/1M 输出；Pro $1.74/$3.48
- 已集成 Claude Code、OpenClaw 等主流 Agent 工具

技术上引入了 token-wise 压缩 + DSA（DeepSeek Sparse Attention）实现超高效长上下文处理，训练数据量从 V3 的 14.8T 翻倍至 33T tokens。在 Agentic Coding 基准上达到开源 SOTA。

**为什么值得深读**：DeepSeek 正在证明开源路线可以在效率和性能上同时比肩甚至超越闭源模型。1M 上下文 + Agent 优化的方向意味着"本地部署顶级 AI Agent"正在从愿景变为现实。

## 2. Simon Willison：GPT-5.5 提示指南 + "人们不渴望自动化"

**来源**：[GPT-5.5 提示指南](https://simonwillison.net/2026/Apr/25/gpt-5-5-prompting-guide/) | ["人们不渴望自动化"](https://simonwillison.net/2026/Apr/24/the-people-do-not-yearn-for-automation/)

Simon Willison 本周连发两篇重磅内容：

**技术面**：OpenAI 为 GPT-5.5 发布了详细的提示工程指南，推荐在多步骤 Agent 任务中先发一条用户可见的短更新来确认收到请求——从"聊天提示"到"Agent 编排"的范式转变。

**人文面**：The Verge 主编 Nilay Patel 提出了"software brain"概念——用算法和数据库理解一切的思维方式。NBC 民调显示 AI 好感度甚至低于移民执法局；Z 世代对 AI 的愤怒从 22% 升至 31%。

**为什么值得深读**："software brain"概念精准描述了技术圈与普通人之间正在扩大的认知鸿沟——这对所有 AI 从业者和创作者都是重要警示。
