## 🔥 今日深度推荐 Deep Reads

### 1. Ghostty 离开 GitHub — Mitchell Hashimoto 的告别信

**来源**: [mitchellh.com](https://mitchellh.com/writing/ghostty-leaving-github) | HN ⬆️3217 💬951（今日第一）

HashiCorp 联合创始人、Vagrant/Terraform 作者 Mitchell Hashimoto 宣布将他的终端模拟器项目 Ghostty 迁离 GitHub。在使用 GitHub 18 年后，他写下了一封感性而坚定的告别信——"GitHub 不再是做严肃工作的地方"。

核心论点：GitHub 频繁的稳定性问题（每天被阻断数小时）已从"偶尔烦人"升级为"严重阻碍生产力"。这不是简单的抱怨，而是一位资深基础设施工程师在用脚投票。他正与多家商业和开源托管方洽谈迁移方案，暂时保留只读镜像。

这篇与 HN 上另外两篇热文形成完美呼应：Armin Ronacher 的 [Before GitHub](https://lucumr.pocoo.org/2026/4/28/before-github/)（⬆️613）回顾了 GitHub 之前的代码协作时代，以及 [We need a federation of forges](https://blog.tangled.org/federation/)（⬆️371）呼吁联邦式代码仓库。加上 Zig 语言 2025 年 12 月迁往 Codeberg，代码托管去中心化正从边缘话题变为主流趋势。

**为什么值得读**: 这不只是一个迁移公告，而是开发者工具生态正在经历一次深层结构性变革的信号弹。

### 2. Zed 1.0 正式发布 — 编辑器新纪元

**来源**: [zed.dev](https://zed.dev/blog/zed-1-0) | HN ⬆️844 💬292

由 Atom 编辑器联合创建者 Nathan Sobo 领衔、100% Rust 从零构建的代码编辑器 Zed 迎来 1.0 里程碑。5 年开发、超百万行代码，终于跨平台稳定发布。

技术亮点：自研 GPU 框架 GPUI 实现 120FPS 渲染、2ms 按键延迟；原生多人实时协作；Agent Panel 支持所有主流 AI 模型；0.6 秒冷启动、222MB 内存 vs VS Code 的 3.5GB。

**为什么值得读**: 代表了一种回归本质的编辑器哲学——极致性能 + AI 原生 + 协作优先。

## 💬 社区热议 Community Buzz

### "你的手机即将不再属于你" — Keep Android Open 运动

**来源**: [keepandroidopen.org](https://keepandroidopen.org/en/) | HN ⬆️1593 💬797

一个名为 "Keep Android Open" 的民间运动正在 HN 上引爆讨论。核心诉求：反对 Google 通过 Play Integrity API 持续收紧 Android 系统的开放性。

**正方**: Android 的开放性正在被温水煮蛙式地剥夺。安装第三方 ROM、root 手机会导致银行 App 拒绝运行——这不是保护用户，是控制用户。

**反方**: Play Integrity 存在的核心理由是安全和反作弊。没有硬件认证，恶意软件可以伪装成任何设备。

**最精彩评论**: "问题不是'设备安全认证是否有价值'，而是'谁有权决定你的设备是否可信'。当这个权力集中在一家公司手里时，它不再是安全功能，而是控制杠杆。"
