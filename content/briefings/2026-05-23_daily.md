## 🔥 今日深度推荐 Deep Reads

### 1. Andrej Karpathy 加入 Anthropic：AI 圈年度人事大新闻

**[TechCrunch 报道](https://techcrunch.com/)** · **[The Technology Express 跟进](https://thetechnologyexpress.com/)** · 个人博客 [karpathy.github.io](https://karpathy.github.io/)

Andrej Karpathy（OpenAI 联合创始人、特斯拉前 AI 总监）本周二晚在 X 发出一条简短动态："Personal update: I've joined Anthropic." 并表示"未来几年 LLM 前沿将特别具有形成性意义"，对回归 R&D 感到兴奋。

**关键细节**：
- 加入 Anthropic **预训练团队**，由前 OpenAI 员工 Nicholas Joseph 领导（该团队负责 Claude 全部大规模训练）
- Karpathy 将在该团队下**组建新的子团队，专注于"用 Claude 自身加速预训练研究"**——"AI 制造下一代 AI"的具体落地
- 此前他经营 Eureka Labs（AI 教育）并发布了超火的 GPT-2 复现教程；今年初他还以"Claude Code 最有影响力的外部用户"身份频繁推荐 Anthropic 工具链

**为什么值得读**：这是去年到今年最具象征意义的 AI 人事变动之一。Axios 称为 Anthropic 在"人才争夺战"中的重大胜利，发生在 SpaceX IPO 与 Anthropic 多轮融资敏感节点。对游戏美术 / AI 内容创作者也有间接含义——Claude/AI 工具的供应方进一步聚拢，工具链的方向决定我们下一个工作流。

### 2. Simon Willison · Datasette Agent 首发：三年工作终于汇合

**[原文链接](https://simonwillison.net/2026/May/21/datasette-agent/)** · 配套插件 [sprites（沙箱执行）](https://simonwillison.net/2026/May/21/datasette-agent-sprites/) · [charts（图表）](https://simonwillison.net/2026/May/21/datasette-agent-charts/)

Simon Willison（Django 共同创始人、前 GitHub）把他三年的 Datasette + LLM 工作终于"组装起来"，发布了 Datasette Agent——一个让你用自然语言对话查询数据库、生成 SQL、可视化结果、并且在沙箱中安全执行 LLM 生成代码的完整方案。

**为什么值得读**：这是一个非常"Simon 式"的工程实践——不追大模型新闻，而是把 LLM 的能力扎扎实实接到一个具体业务场景里（数据查询）。他对 sandboxing 的设计思路（sprites 插件）尤其值得读，对所有想做"LLM 调本地工具但又怕被搞破坏"的人都是参考案例。
