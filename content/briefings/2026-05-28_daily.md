## 🔥 今日深度推荐 Deep Reads

### 1️⃣ Anthropic 和 OpenAI 找到 PMF

**作者**: Simon Willison
**来源**: [simonwillison.net](https://simonwillison.net/2026/May/27/product-market-fit/)

Simon Willison 这篇短评抓住了一个关键拐点：**强烈传闻 Anthropic 即将迎来首个盈利季度**，而企业用户开始按 API 价大规模付费——员工大量使用导致公司 LLM 账单飙升，但这恰恰是 PMF 最强的信号。

Simon 的判断是：两家头部 LLM 公司**都已经过了"找到 PMF"这一步**，正在加速 ramp up。这跟两年前"AI 公司烧钱无尽头、商业化遥遥无期"的叙事完全反过来了。配合本周另一条新闻——Cognition（Devin 母公司）ARR 达 4.92 亿美元、估值 8 个月翻倍至 25B——AI 编码这条线正在跑出真正的现金流。

**为什么值得读**：作为长期跟踪 AI 工具的资深开发者，Simon 的判断比媒体报道更克制也更准。如果 PMF 已成立，下一波焦点就是**毛利率改善 + 企业级护城河**，而不是再烧一个数量级的钱去抢用户。

### 2️⃣ Andrej Karpathy 加入 Anthropic（5/19 官宣）

**来源**: [搜狐综合](https://www.sohu.com/a/1025827357_122500756) ｜ TheNewJerseyTimes

5月19日，Karpathy 在 X 官宣加入 Anthropic，进入 Nick Joseph 领导的预训练团队，负责组建一支用 Claude 自身加速预训练研究的新队伍。

**信号意义**：

- 离开 OpenAI 近 1.5 年后，他选择回到牌桌——不是回 OpenAI，不是去 Google，而是 Anthropic
- 在 3月20日的播客里他曾说"对游离在前沿实验室之外感到紧张，警惕完全被单一闭源实体所掌控"，这句话现在有了答案
- 教育领域工作（Eureka Labs）"暂时按下暂停键"
- 官宣金句："LLM 的前沿将会特别具有塑造性"

**为什么值得读**：当一位顶级 researcher 公开评估"现在是不是值得回到前沿"，他的脚投票本身就是一份对 LLM 未来 2-3 年走向的判断书。

## ✨ 今日寄语

> **「LLM 的前沿将会特别具有塑造性。」** —— Andrej Karpathy 加入 Anthropic 时的官宣
