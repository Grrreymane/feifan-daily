## 🔥 今日深度推荐 Deep Reads

### 1. Anthropic 发布 Project Glasswing：AI 网络安全的拐点时刻

**来源**: [Anthropic 官方博客](https://www.anthropic.com/glasswing) | [Wired 深度报道](https://www.wired.com/story/anthropic-mythos-preview-project-glasswing/) | [CyberScoop](https://cyberscoop.com/project-glasswing-anthropic-ai-open-source-software-vulnerabilities/)

Anthropic 正式发布了 Claude Mythos Preview 模型，并联合 Apple、Microsoft、Google、AWS、Nvidia、CrowdStrike、Linux Foundation 等 12 家巨头 + 40+ 组织成立了 Project Glasswing 网络安全联盟。

Mythos 已在所有主流操作系统和浏览器中发现了数千个高危零日漏洞——包括 OpenBSD 中存在 27 年的 bug 和 FFmpeg 中存在 16 年的漏洞。Anthropic 承诺投入 1 亿美元使用额度 + 400 万美元直接捐赠给开源安全组织。

这是历史上第一次——一家 AI 公司明确表示自己的模型"太危险不能公开发布"，并组织了一个结构化的防御联盟。Anthropic 同时正因拒绝为五角大楼移除自主武器限制而面临政治压力，在如此环境下仍坚守安全底线。

### 2. Latent Space: 极限 Harness Engineering——百万行代码、零人类代码、零人工审核

**来源**: [Latent Space](https://www.latent.space/p/harness-eng) | [YouTube](https://www.youtube.com/watch?v=CeOXx-XTYek) | [OpenAI 博文](https://openai.com/index/harness-engineering/)

OpenAI Frontier 团队的 Ryan Lopopolo 揭开了 OpenAI 内部第一个"暗工厂"的面纱：5个月，100万行代码，0行人类手写代码，0行经过人工审核后合并。"Harness Engineering"正式取代 Prompt Engineering 和 Context Engineering，成为 2026 年最重要的软件工程范式。

核心洞察："唯一真正稀缺的是人类注意力"——token 可以无限并行，人类不行。前1.5个月比自己写代码慢10倍，但坚持投入后产出超过任何单个工程师。

## 💬 社区热议 Community Buzz

### "Vibe Coding 邪教太疯狂了" — Bram Cohen 开炮

**来源**: [Bram Cohen 博客](https://bramcohen.com/p/the-cult-of-vibe-coding-is-insane) | HN ⬆️591 💬488

BitTorrent 发明者 Bram Cohen 痛批 Anthropic 的 Claude Code 团队将"dogfooding"推到了极端——团队成员坚持"纯 vibe coding"，拒绝查看 AI 生成的代码。结果外部开发者一眼发现了大量代码重复问题。

与此同时，Claude Code 在复杂工程任务中变得不可用的 GitHub Issue（#42796）登上 HN 本周第 4 热帖（⬆️1285 💬712），Anthropic 不再允许 Claude Code 用户使用 OpenClaw 也引发巨大争议（⬆️1095 💬829）。

最精彩的评论："Bad software is a decision you make. You need to own it." 这与 Ryan Lopopolo 的 Harness Engineering——同样 0% 人工审核但通过严格架构约束保证质量——形成了有趣的对比。
