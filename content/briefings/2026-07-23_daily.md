### 1. [Anthropic 5 月营收年化跑到 470 亿美元：Menlo Ventures 合伙人"25 年从未见过这种增长速度"](https://techcrunch.com/video/menlo-ventures-matt-murphy-explains-why-anthropic-is-winning-and-its-not-the-model/)

Anthropic 从 2025 年 90 亿美元 ARR 一路冲到 2026 年 5 月的 470 亿美元。Menlo Ventures 合伙人 Matt Murphy 在 TechCrunch 视频访谈中直言："25 年投资生涯中从未见过——互联网、移动、云都没有。"更关键的观点：Anthropic 的护城河已经**不再是模型能力本身**，而是**企业分发**——它比 OpenAI 更早在 Coding / Agent / 企业工作流三条线上做到了"深度垂直嵌入"，而不仅仅是通用 API。

**为什么值得读**：这是判断 2026 年 AI 生态格局的关键锚点。对于游戏美术从业者而言，理解"AI 巨头的护城河从模型转向分发"，也意味着未来更多游戏工作室级别的 AI 工具集成会走 Anthropic 这条路径（Claude Code 已经是先兆），而非纯 API 调用。

### 2. [Game Maker's Toolkit：《侠盗猎车手》如何把整座城市塞进 PS2 内存](https://www.youtube.com/watch?v=z5AoBAoxpNU) 🎮

Mark Brown 最新一期 GMTK 长视频，26 分钟深挖 Rockstar 用什么技术把《GTA III》《Vice City》《圣安地列斯》塞进只有 32MB 内存的 PS2。核心讲解 Streaming（流式加载）、LOD 分级、地图分块、纹理压缩、内存池等所有关键美术/技术管线。Brown 的招牌是**把工程约束翻译成美术决策**——比如为什么远处建筑必须是"棱柱化"造型、为什么天气系统要设计成"雾气覆盖切换"，全都是内存优化倒逼出的美学。

**为什么值得看**：这是 3D 场景美术+TA 从业者的年度必看深度视频。当下 UE5 / Unity 项目动辄十几 GB，反过来看 2004 年前后是怎么用"美术+工程"共同榨干硬件的，对现代项目的性能优化仍然有大量可迁移思路。
