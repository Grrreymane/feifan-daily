## 🔥 今日深度推荐 Deep Reads

### 1. Karpathy "AI 精神病" + Sequoia AI Ascent 2026 对谈

**来源**：[CSDN 中文解读](https://blog.csdn.net/loongggdroid/article/details/160721414)

Andrej Karpathy 近日做客播客并在 Sequoia AI Ascent 2026 发表对谈，信息密度极高。核心观点：

- **"AI 精神病"**：他坦言自己患上了一种新型焦虑——"Token 没用完就觉得浪费了"，这反映了整个行业对 AI 能力的贪婪追逐
- **代码正在消失**：自然语言正在成为新的编程接口，传统代码的角色被重新定义
- **Agent 重构软件生产**：不是"AI 辅助写代码"，而是 Agent 作为核心主体，人类作为审核者
- **"从没像现在这样觉得自己落后"**：即便是 AI 领域的顶尖人物，也感受到了技术加速带来的认知压力

**为什么值得读**：这是 AI 领域最具影响力的人物之一的年度核心表达。无论你是开发者、美术、还是产品经理，他描述的"软件 3.0"范式转变都直接影响你的工作方式。

### 2. Graph-Enhanced RAG：超越向量搜索的架构模式

**来源**：[VentureBeat](https://venturebeat.com/orchestration/architectural-patterns-for-graph-enhanced-rag-moving-beyond-vector-search-in-production)

这篇深度技术文章剖析了 RAG（检索增强生成）在企业落地时，纯向量检索的三大局限：

- **语义漂移**：向量相似 ≠ 逻辑相关，在高度互联的数据中（供应链、医疗、法律），纯语义匹配会丢失关键关系
- **图增强架构**：引入知识图谱 + 向量检索 + 结构化查询的三层架构
- **生产实践**：具体的架构模式和实施路径

**为什么值得读**：如果你在做 AI 应用（尤其是企业级 RAG），这是目前最好的工程化落地指南之一。

### 1. Julia Evans：离开 Tailwind，学习组织 CSS

**来源**：[Julia Evans 博客](https://jvns.ca/blog/2026/05/15/moving-away-from-tailwind--and-learning-to-structure-my-css-/) | [HN 讨论 ⬆️626 💬352](https://news.ycombinator.com/item?id=48158400)

Julia Evans 分享了她从 Tailwind CSS 回归原生 CSS 的经历。Tailwind 虽高效，但让她对 CSS 本身的理解停滞。过去 10 年 CSS 已经解决了很多老问题，但这些进步被 Tailwind 的抽象层遮蔽了。

HN 上引爆了"工具 vs 基础"的经典辩论。精彩评论：「我们这代开发者用 jQuery 绕过了学 DOM 的痛苦，下一代用 Tailwind 绕过了学 CSS 的痛苦——然后我们都在面试时被反噬了。」
