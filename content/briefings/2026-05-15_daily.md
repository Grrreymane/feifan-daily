## 💬 社区热议 Community Buzz

### 1. Linux 游戏性能超越 Windows：因为 Windows API 正成为 Linux 内核特性

📎 [XDA Developers](https://www.xda-developers.com/linux-gaming-is-getting-faster-because-windows-apis-are-becoming-linux-kernel-features/) | HN 930⬆️ 570💬（本周最热）

Linux 6.18 新增了 NT 同步原语（NT synchronization primitives）内核驱动，Wine/Proton 游戏性能暴涨。不再通过用户态模拟 Windows 同步机制，而是内核态原生支持。

**精彩观点**：
- "这不是抄袭Windows，而是Linux内核做了它一直擅长的事——把跑在用户态的低效模拟下沉到内核态原生支持"
- "Valve 正在做微软没做到的事——让你在非 Windows 上玩 Windows 游戏比在 Windows 上玩还流畅"
- 质疑："Wine/Proton 的成功恰恰证明了原生 Linux 游戏生态的失败"
